# @auth0/ai

`@auth0/ai` is a framework for building generative AI agents that operate on behalf of a
person.



auth0-ai-js!

## Running examples

1. Install depedencies for the workspace

   ```sh
   $ npm install
   ```

2. Run [Turbo](https://turbo.build/)

   ```sh
   $ npm run build
   ```

3. Follow example instructions
